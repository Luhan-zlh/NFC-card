# 给凯玟的卡片 — 部署与使用说明

这是一个纯静态网页（HTML/CSS/JS，无需服务器），用于 NFC 卡片碰一下后打开。
文件结构：

```
nfc_redirect_site/
├── index.html        网页结构
├── css/style.css      视觉样式（深色星空风）
├── js/data.js          【你要改的内容都在这里】名字/日期/时间线/小纸条/留言本链接
├── js/script.js        逻辑代码（一般不用改）
└── images/             把你们的照片放这里，然后在 data.js 里填相对路径
```

---

## 第一步：改内容（不需要写代码）

打开 `js/data.js`，按提示替换：

1. `herName`、`anniversaryDate` — 已经帮你填好了（凯玟 / 2025-08-04）
2. `timeline` 数组 — 把占位的三条换成你们真实的重要日期和故事，想加几条都可以，复制一段花括号 `{...}` 粘贴修改即可
3. `loveNotes` 数组 — 先用占位的 4 句，你有空了随时替换成自己写的话
4. `sharedNotebookUrl` — 见下面"共享留言本"部分

把照片放进 `images/` 文件夹（建议提前压缩到 500KB 以内一张，加载更快），然后在对应时间线条目里把 `img: ""` 改成 `img: "images/你的照片文件名.jpg"`。

---

## 第二步：共享留言本（腾讯文档，双方都能写）

1. 打开 https://docs.qq.com ，登录（微信或QQ号即可）
2. 新建一个"在线文档"，标题随便取，比如"我们的悄悄话"
3. 点右上角"分享" → 设置为"**互联网上获得链接的人可编辑**"
4. 复制生成的链接，粘贴到 `data.js` 里的 `sharedNotebookUrl` 那一行引号中间
5. 把这个链接也发给她一份（或者放进网页她自己点开就能看到）

这样你们俩以后随时可以打开这个链接加新的话，不需要碰代码、不需要重新部署网页。

---

## 第三步：部署到 Gitee Pages（国内可稳定访问）

### 3.1 注册 Gitee 账号 + 实名认证（你需要亲自操作，越早做越好）

1. 打开 https://gitee.com ，注册账号
2. 登录后，右上角头像 → "账号设置" → "实名认证"
3. 按提示用支付宝或身份证完成认证（通常几分钟到1天内通过）

**这一步是整个流程里唯一有等待时间的环节，建议现在就去做。**

### 3.2 新建仓库

1. Gitee 首页右上角 "+" → "新建仓库"
2. 仓库名称随意，比如 `nfc-card`
3. **是否开源：选"私有"**（保护隐私，别人搜不到、看不到源码）
4. 创建完成

### 3.3 上传代码

在这个文件夹（`nfc_redirect_site`）里执行：

```powershell
git init
git add .
git commit -m "first version"
git remote add origin https://gitee.com/你的用户名/nfc-card.git
git push -u origin master
```

（如果提示需要登录，用你的 Gitee 账号密码或者按提示生成的私人令牌）

### 3.4 开启 Gitee Pages

1. 进入仓库页面，点上方 "服务" → "Gitee Pages"
2. 部署分支选择 `master`，部署目录留空（代表根目录）
3. 点击"启动"
4. 稍等片刻，会生成一个网址，形如：
   `https://你的用户名.gitee.io/nfc-card/`

**这个网址就是最终要写入 NFC 芯片的链接。**

> 注意：Gitee Pages 私有仓库的 Pages 页面本身默认也需要登录才能访问（这是 Gitee 的隐私保护机制）。
> 如果发现她打不开，去 "Gitee Pages" 设置里找 "**是否开启强制认证**" 选项，关闭它（这样任何人拿到链接都能看网页内容，但仓库源码依然是私有、看不到的，两者互不影响）。

### 3.5 以后要更新内容怎么办？

改完 `data.js` 之后：

```powershell
git add .
git commit -m "update content"
git push
```

然后回到 Gitee 仓库页面的 Pages 设置里，点一下"更新"按钮（Gitee Pages 不会自动重新构建，需要手动点一次）。

---

## 第四步：把链接写入 NFC 卡片

1. 手机（iOS 或 Android 均可）应用商店下载 **"NFC Tools"**（免费）
2. 打开 App → "写入" → "添加一条记录" → 选择 "URL/URI"
3. 输入你的 Gitee Pages 网址（必须是 `https://` 开头）
4. 点击"写入" → 把手机贴近卡片背面 NFC 芯片位置 → 完成

**重要提醒（iOS 兼容性）**：
- 网址必须是 `https://`，不能是 `http://`，否则 iPhone 不会自动弹出识别
- 写入前建议先用另一部手机测试卡片本身是否正常识别（碰一下能不能弹出网址）
